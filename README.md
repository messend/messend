### Hey, soy Messen 👋

Hace no mucho no sabía nada de programación.  
Hoy construyo aplicaciones full-stack con React y Python.

Eso pasó porque decidí apostar fuerte y entrar al bootcamp de **4Geeks Academy** — uno de los más exigentes de España.  
No fue fácil, pero cada proyecto, cada bug, cada deploy fallido me enseñó algo.

Ahora estoy buscando mi primer rol como **Junior Full-Stack Developer** donde seguir creciendo y aportar desde el día uno.

> 🎯 **Goal:** Land my first dev job & never stop building 🚀
